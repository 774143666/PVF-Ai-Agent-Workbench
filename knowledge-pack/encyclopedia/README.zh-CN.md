# PVF 百科

这里放纯结论百科。每个文件只回答：

- 这个系统或文件是什么。
- 常见路径和 registry 是什么。
- 关键标签或字段是什么意思。
- 能怎么用。
- 哪些地方必须验证。

不要写来源报告、实验路径、旧资料路径或证明过程。

## 状态

- `默认可用`：可以直接作为 Agent 的默认理解。
- `需验证`：只能当线索，必须查目标 PVF、客户端或实机。
- `禁用`：不能照做。
