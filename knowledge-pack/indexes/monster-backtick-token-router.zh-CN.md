# Monster .mob 反引号 Token 路由

本索引记录只读目标 PVF 的 monster `.mob` 中实际观察到的方括号形反引号 token。token 必须结合父字段和所在块解释；同名 token 在不同父字段中不得自动共享语义。

## 使用边界

- token 不是独立字段，不能脱离父标签解释。
- 能力、类别、目标、难度、状态和对象 token 必须保留原上下文。
- 本索引只证明目标 PVF 中观察到该 token，不证明运行效果。
