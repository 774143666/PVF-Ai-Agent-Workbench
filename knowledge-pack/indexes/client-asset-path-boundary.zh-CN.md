# 客户端资源路径预览边界

状态：默认可用

## 可写入报告的结论

- PVF 中出现的资源路径。
- 资源路径来自哪个 PVF 文件类型或 planner 输出。
- 客户端目录中是否发现候选文件或容器。
- 未命中路径、疑似外部容器、路径大小写或别名风险。

## 不可写入为事实的结论

- 图层显示正确。
- 动画帧序列正确。
- UI 控件可点击。
- 地图贴图、机关、怪物动作、光环、时装、宠物特效正常。
- 音频播放正常。

## 写资源前必须补齐

1. 明确目标客户端。
2. 明确 NPK/ImagePacks2 目标容器。
3. 生成可审阅的最小资源变更清单。
4. 保留原客户端资源备份。
5. 部署后做实机视觉或音频验证。
