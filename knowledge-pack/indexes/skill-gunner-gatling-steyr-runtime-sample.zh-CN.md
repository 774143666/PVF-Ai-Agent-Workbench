# 男枪手 Gatling / Steyr 运行样本边界

状态：需验证

用途：给 Agent 处理连射、按键延长、爆炸次数、TP/特性覆盖类技能时提供样本级规则。以下是样本结论，不是全版本通用定律。

## 角色目录

- `skill/gunner/...` 与 `skill/atgunner/...` 属于不同角色或职业分支目录，不是觉醒阶段。
- `at` 前缀按独立角色/分支 token 处理；解析技能前必须回到目标 PVF 的角色入口、职业技能 registry 和技能树。

## Gatling 样本

| 项 | 样本结论 |
| --- | --- |
| 基础技能 | `skill/gunner/gatling.skl` |
| TP/特性候选 | `skill/gunner/gatlingex.skl`，目标 PVF 仍需 registry 或技能树确认。 |
| `[static data]` 静态 0 | 样本中参与按住或连按 X 时的最大子弹数计算。 |
| `[level info]` 动态 2 | 样本中表现为发射时间上限，单位按毫秒理解。 |
| 最大子弹数样本公式 | `static[0] * (levelInfo[2] / 1000)`。 |
| `[level info]` 动态 1 | 样本中表现为发射间隔或射速相关列；降低会明显提高射速。 |
| 非按住/非连按 | 可能走另一套基础发射行为，不能直接套最大子弹公式。 |
| 高上限观察 | 200 发只记录为样本观察，不能当跨版本硬上限。 |

## Steyr / 反坦克炮样本

| 项 | 样本结论 |
| --- | --- |
| 基础技能 | `skill/gunner/steyrheavyrifle.skl` |
| TP/特性候选 | `skill/gunner/steyrheavyrifleex.skl`，目标 PVF 仍需 registry 或技能树确认。 |
| 爆炸次数 | 基础与 Ex/TP 文件同步后，样本中不点 TP 和点 TP 可保持一致。 |
| 误判风险 | 曾出现把动作/取出速度相关列误当爆炸大小的情况；字段说明不能替代实机验证。 |
| 视觉大小 | 爆炸视觉大小不只由 `.skl` 数值保证；视觉不变时继续查 passiveobject、ANI/ATK 或客户端资源。 |

## 实机验证最小集

1. 不点 TP：普通按键、按住或连按键。
2. 点 TP/特性：普通按键、按住或连按键。
3. 观察最大次数、射速、动作速度、爆炸次数、视觉范围。
4. 若只改基础 `.skl` 后点 TP 行为不一致，立即检查对应 `*ex.skl` 或技能树指向的特性技能。
